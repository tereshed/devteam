## Канон (SSOT)

| Тема | Решение |
|------|---------|
| Источник данных | Только **`AgentModel`**. **`AgentCard`** не ходит в **`TeamRepository`** и не читает **`TeamModel`**. |
| Локализация | Все пользовательские строки через **`AppLocalizations.of(context)!`** внутри **`build`**. **Запрещено** обязательный параметр **`AppLocalizations`** в публичном конструкторе (**как 12.6**). |
| Роль | Только **`agentRoleLabel`** из **`agent_role_display.dart`**. |
| Модель LLM | Пустое / **`null`** → ключ **`teamAgentModelUnset`** (**13.1**). |
| Имя агента | Пустое после **`trim`** → **`teamAgentNameUnset`** (**13.1**). |
| **`code_backend`** | Опциональная строка: если после **`trim`** пусто — строку не показывать (как сейчас **`_optionalLine`**). Значение с wire выводить **как есть** (строка из **`AgentModel`**). В **13.2** **не** вводить локальный маппинг подписей **`code_backend`** и **не** подключать новый общий хелпер отображения, даже если он появится в смежном PR; подключение маппинга — **отдельная задача**. |
| **`prompt_name`** | Та же опциональность, что **`codeBackend`**. В названии строки README **13.2** не перечислен — но в **13.1** карточка агента уже показывает **`promptName`**; при выносе в **`AgentCard`** строку **сохранить**, чтобы не регрессировать UX вкладки «Команда». |
| Активность (**on/off**) | Отображение **`isActive`**: **`Chip`** + **`Semantics(label: …)`** — **`teamAgentActive`** / **`teamAgentInactive`** (**13.1**). **Запрещены** интерактивные **`Switch`/`Checkbox`** в **13.2** (мутации — **13.3**). Связка **`Semantics` + `Chip(label: Text(...))`** — намеренный паттерн a11y (**README** Sprint 13, блок «Решения по дизайну»); при **`excludeSemantics`** на **`Chip`** не дублировать озвучивание. |
| Навигация | **`AgentCard`** не импортирует **`go_router`**. При необходимости тапа — **`onTap`** снаружи (**опциональный** параметр, по умолчанию **`null`**). |
| Тема | **`Theme.of(context)`**, **`ColorScheme`**, **`TextTheme`**; **`Color.withValues(alpha: …)`**, не **`withOpacity`**. |

## Контракт пропсов (минимум)

| Параметр | Тип | Назначение |
|----------|-----|------------|
| **`key`** | **`Key?`** | Через **`super.key`** в конструкторе; для **`ListView`**, **`ValueKey(agent.id)`** в тестах **`find.byKey`**. |
| **`agent`** | **`AgentModel`** | Данные агента. |
| **`onTap`** | **`VoidCallback?`** | По умолчанию **`null`**: без **`InkWell`**, без семантики кнопки на корне. Если не **`null`** — оборачивать зону тапа в **`InkWell`** (под уже существующим **`Material`** дерева — **`Scaffold`**, **`Card`** и т.д.; **не** добавлять лишний **`Material`** только ради сплеша) и задавать **`Semantics`** кнопки (**как 12.6**). |

Опциональное расширение API (**например `dense`**) — только отдельным согласованием в PR; дефолты не должны ломать **`TeamScreen`**.

## Сценарии widget-тестов (`agent_card_test.dart`)

Базовая обёртка: **`MaterialApp`** с **`locale: const Locale('ru')`**, делегаты **`AppLocalizations`** — **точный эталон:** **`frontend/test/features/tasks/presentation/widgets/task_card_test.dart`** (паттерн делегатов / локали; не путать с **`features/team/`**).

- Активный агент — видимы **`teamAgentActive`** (текст и/или семантика).
- Неактивный — **`teamAgentInactive`**.
- **`name`** пустой — подстановка **`teamAgentNameUnset`**.
- **`model == null`** — **`teamAgentModelUnset`**.
- С **`codeBackend`** / **`promptName`** — подстроки на экране; без них — отсутствуют.
- **`onTap != null`** — **`Semantics`** с **`button: true`** (или эквивалент **12.6**); тап вызывает коллбек; **`InkWell`** зоны карточки (вокруг основного **`Padding(16)`**) — один; **`find.byType(InkWell)`** по всему поддереву **`AgentCard`** не использовать — у **`Chip` (Material 3)** может быть свой **`InkWell`** для сплеша.
- **`onTap == null`** — на корне карточки **не** задавать семантику кнопки **и** нет **`InkWell`** зоны тапа карточки (тот же предикат «**`InkWell`** + **`Padding(all: 16)`**» — **`findsNothing`**).
