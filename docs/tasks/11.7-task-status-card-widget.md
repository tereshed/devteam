## Контекст

В **`ConversationMessageModel`**: **`linkedTaskIds`**, **`metadata`**.

Событие WebSocket: тип **`'task_status'`**, класс **`WsTaskStatusEvent`**, вариант **`WsServerEvent.taskStatus`** (**`frontend/lib/core/api/websocket_events.dart`**).

**Поля `WsTaskStatusEvent` (Dart, полный перечень):** **`ts`**, **`v`**, **`projectId`**, **`taskId`**, **`parentTaskId`**, **`previousStatus`**, **`status`**, **`assignedAgentId`**, **`agentRole`**, **`errorMessage`**.

**Из этого набора карточка 11.7 потребляет из снимка/пропсов только:** **`taskId`**, **`status`**, **`errorMessage`**, **`agentRole`** (последнее — после приведения к **`TaskCardAgentRole?`** на границе контроллера). **`previousStatus`**, **`parentTaskId`**, **`assignedAgentId`**, **`ts`**, **`v`**, **`projectId`** в **11.7** **не** используются.

Карточка **не** дублирует **`ChatMessage`**.

## Публичный API (ориентир)

```dart
class TaskStatusCard extends StatelessWidget {
  const TaskStatusCard({
    super.key,
    required this.taskId,
    this.title,
    required this.status,
    this.errorMessage,
    this.agentRole,
    this.onOpen,
  });

  final String taskId;
  final String? title;
  final String status;
  final String? errorMessage;
  final TaskCardAgentRole? agentRole;
  final ValueChanged<String>? onOpen;
}
```

**`Key`:** родитель передаёт **`key: ValueKey(taskId)`**. В debug: **`assert(key is ValueKey<String> && (key as ValueKey<String>).value == taskId)`**.

Интерактив при **`onOpen != null`:** **`InkWell`** в составе **`Material`** (не **`GestureDetector`** без клавиатурной семантики).

## Компоновка карточки (LTR baseline)

Иконка — **leading** слева от текста; в **RTL** — через **`EdgeInsetsDirectional`**.

```
┌─────────────────────────────────────────────────────────┐
│ [icon]  Title (1–2 lines, max)                         │
│         localized_status  ·  localized_agent_role      │
│         (блок ошибки только при failed)               │
│         error_message (maxLines: 3)                   │
└─────────────────────────────────────────────────────────┘
```

- **Padding** содержимого: **`EdgeInsets.all(16)`**. Если у **`ChatMessage`** (**11.6**) другой внутренний отступ — в одном PR выровнять на **`16`** **или** вынести **`kChatBubblePadding`** в shared и использовать **одно** значение для пузыря и карточки.
- **`minHeight`** не задавать.
- Строка 2: статус; при **`agentRole != null`** — **` · `** + локализованная роль; иначе **без** **`·`** и без пустого места.
- **`errorMessage`:** только **`failed`**; отступ **`SizedBox(height: 8)`** над ошибкой. **`Divider`** перед ошибкой **запрещён** в **11.7**.
- Корень: **`Material`** (**`surfaceContainerLow`**, elevation **0**). Вариант **`Card`** в **11.7** **запрещён** (один тип корня).
