# 3.8 Unit-тесты: TaskService

**Sprint:** 3 — CRUD API для Tasks
**Файл:** `backend/internal/service/task_service_test.go`
**Статус:** ✅ Выполнена

---

## Описание

Unit-тесты для `TaskService` — ключевого сервиса Sprint 3. Покрывают все 11 методов интерфейса, state machine переходы, ABAC, optimistic locking, валидацию и edge cases.

---

## Реализация: факт

- **49 unit-тестов** (спецификация в 3.3 предполагала 46 — реализовано 3 дополнительных)
- **767 строк** тестового кода
- **4 mock-зависимости:** `mockTaskRepository`, `mockTaskMessageRepository`, `mockTaskProjectService`, `mockTaskTeamService`
- **Harness:** `newTaskServiceHarness()` — фабрика для быстрого создания моков и сервиса
