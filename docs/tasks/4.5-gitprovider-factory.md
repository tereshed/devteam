# 4.5 Фабрика: NewGitProvider(providerType, credentials)

**Sprint:** 4 — Git Provider Integration (Backend)
**Файл:** `backend/pkg/gitprovider/factory.go`
**Статус:** ✅ Выполнена

---

## Описание

Реализовать конкретную структуру `DefaultFactory`, удовлетворяющую интерфейсу `Factory`, определённому в `provider.go`:

```go
// provider.go (уже существует):
type Factory interface {
    Create(providerType string, creds Credentials) (GitProvider, error)
}
```

Фабрика — единая точка создания экземпляров `GitProvider`. Вызывающий код (`ProjectService`, `OrchestratorService`) получает фабрику через конструктор (Dependency Injection), вызывает `Create(...)` и работает с абстракцией `GitProvider`, не зная о конкретных реализациях.

## Что нужно реализовать

### 1. Структура `DefaultFactory`

```go
type DefaultFactory struct{}
```

Структура без полей — вся логика в маршрутизации по `providerType` и вызове конструкторов.

### 2. Конструктор `NewFactory`

```go
func NewFactory() Factory {
    return &DefaultFactory{}
}
```

Возвращает интерфейс `Factory`. Вызывается в `cmd/api/main.go` при сборке зависимостей.

### 3. Метод `Create`

```go
func (f *DefaultFactory) Create(providerType string, creds Credentials) (GitProvider, error)
```

**Логика:**

1. **Нормализация входа** — привести `providerType` к нижнему регистру, обрезать пробелы: `strings.ToLower(strings.TrimSpace(providerType))`.

2. **Маршрутизация по типу:**

| `providerType` (после нормализации) | Конструктор | Файл |
|---|---|---|
| `"github"` | `NewGitHubProvider(creds)` | `github.go` |
| `"local"` | `NewLocalGitProvider(creds)` | `local.go` |

3. **Неизвестный тип** — вернуть sentinel-ошибку `ErrUnknownProvider` с оборачиванием через `%w`:
   ```go
   return nil, fmt.Errorf("%w: %q", ErrUnknownProvider, providerType)
   ```
   - `ErrUnknownProvider` добавлен в `provider.go` (sentinel errors).
   - Оригинальный `providerType` (до нормализации) включается в сообщение, чтобы в логах был виден точный вход.
   - Вызывающий код может программно обработать ошибку: `errors.Is(err, gitprovider.ErrUnknownProvider)`.

4. **Пустой `providerType`** — после нормализации станет `""`, попадёт в `default` → та же ошибка `ErrUnknownProvider`.

### 4. Compile-time проверка интерфейса

```go
var _ Factory = (*DefaultFactory)(nil)
```

## Существующие конструкторы провайдеров

Фабрика вызывает уже реализованные конструкторы:

**GitHubProvider** (`github.go:29-37`):

```go
func NewGitHubProvider(creds Credentials) *GitHubProvider {
    ts := oauth2.StaticTokenSource(&oauth2.Token{AccessToken: creds.Token})
    httpClient := oauth2.NewClient(context.Background(), ts)
    client := github.NewClient(httpClient)
    return &GitHubProvider{
        LocalGitCLI: LocalGitCLI{creds: creds},
        client:      client,
    }
}
```

**LocalGitProvider** (`local.go:20-22`):

```go
func NewLocalGitProvider(creds Credentials) *LocalGitProvider {
    return &LocalGitProvider{LocalGitCLI: LocalGitCLI{creds: creds}}
}
```

## Примерная реализация

```go
package gitprovider

import (
    "fmt"
    "strings"
)

var _ Factory = (*DefaultFactory)(nil)

// DefaultFactory creates GitProvider instances by provider type string.
type DefaultFactory struct{}

// NewFactory returns a Factory for creating GitProvider instances.
// Inject into services via constructors (DI).
func NewFactory() Factory {
    return &DefaultFactory{}
}

// Create returns a GitProvider for the given providerType and credentials.
// The providerType is normalized (trimmed and lowercased) before matching.
// Supported types: "github", "local". Returns ErrUnknownProvider otherwise.
func (f *DefaultFactory) Create(providerType string, creds Credentials) (GitProvider, error) {
    switch strings.ToLower(strings.TrimSpace(providerType)) {
    case "github":
        return NewGitHubProvider(creds), nil
    case "local":
        return NewLocalGitProvider(creds), nil
    default:
        return nil, fmt.Errorf("%w: %q", ErrUnknownProvider, providerType)
    }
}
```

## Архитектурные ограничения

1. **Нет зависимости от `internal/models`** — фабрика принимает `string`, а не `models.GitProvider`. Преобразование `string(project.GitProvider)` — на стороне вызывающего.

2. **Нет глобальных переменных** — фабрика создаётся явно через `NewFactory()` и передаётся через конструкторы (правило 2.1.2 из `backend.mdc`).

3. **Пакет `pkg/gitprovider`** — не импортирует ничего из `internal/`.

4. **Credentials уже расшифрованы** — фабрика не занимается шифрованием/расшифровкой (Sprint 4.7).

## Валидация входных данных

| Вход | Результат |
|---|---|
| `providerType = "github"`, `creds.Token = "ghp_xxx"` | `*GitHubProvider`, `nil` |
| `providerType = "GitHub"` (любой регистр) | `*GitHubProvider`, `nil` (после ToLower) |
| `providerType = "  github  "` | `*GitHubProvider`, `nil` (после TrimSpace) |
| `providerType = "local"`, `creds = Credentials{}` | `*LocalGitProvider`, `nil` |
| `providerType = "gitlab"` | `nil`, `ErrUnknownProvider` (wrapping: `unknown provider type: "gitlab"`) |
| `providerType = ""` | `nil`, `ErrUnknownProvider` (wrapping: `unknown provider type: ""`) |
| `providerType = "bitbucket"` | `nil`, `ErrUnknownProvider` (wrapping: `unknown provider type: "bitbucket"`) |

### Обязательные тест-кейсы

| # | Тест-кейс | Описание |

| 3 | `TestFactory_Create_CaseInsensitive` | `Create("GitHub", creds)` и `Create("GITHUB", creds)` — без ошибки, тип `*GitHubProvider` |
| 4 | `TestFactory_Create_Trimmed` | `Create("  github  ", creds)` — без ошибки, тип `*GitHubProvider` |
| 5 | `TestFactory_Create_UnknownType` | `Create("gitlab", creds)` → `nil`, `errors.Is(err, ErrUnknownProvider)` |
| 6 | `TestFactory_Create_EmptyType` | `Create("", creds)` → `nil`, `errors.Is(err, ErrUnknownProvider)` |
| 7 | `TestFactory_ImplementsInterface` | Compile-time: `var _ Factory = (*DefaultFactory)(nil)` |
| 8 | `TestNewFactory_ReturnsFactory` | `NewFactory()` возвращает не-nil |

### Паттерн: Table-Driven Tests

**Импорты:** тесты используют `github.com/stretchr/testify/assert` (уже есть в `go.mod` проекта).

**Проверка типа через `assert.IsType`:** Фабрика возвращает интерфейс `GitProvider`. Мы проверяем конкретный тип возвращаемого значения (`*GitHubProvider`, `*LocalGitProvider`), а не поведение методов `ProviderType()`/`SupportsPullRequests()` — это ответственность тестов самих провайдеров (`github_test.go`, `local_test.go`). `assert.IsType` гарантирует, что фабрика вернула нужную структуру.

**Проверка ошибок через `errors.Is`:** Для error-кейсов используем `errors.Is(err, ErrUnknownProvider)` вместо `assert.Contains(err.Error(), ...)` — программная проверка sentinel error, а не парсинг строки.

**Безопасность токена в тестах:** `NewGitHubProvider` создаёт `oauth2.NewClient` без сетевых запросов при инициализации — фейковый токен `"test-token"` безопасен. Если в будущем конструктор добавит валидацию токена (пинг API), тесты фабрики начнут падать. На этот случай конструктор можно будет обернуть в `interface` или вынести HTTP client в параметр.

```go
package gitprovider

import (
    "errors"
    "testing"

    "github.com/stretchr/testify/assert"
)

func TestFactory_Create(t *testing.T) {
    f := NewFactory()
    tests := []struct {
        name         string
        providerType string
        wantConcreteType interface{}
        wantErr      bool
    }{
        {"github", "github", &GitHubProvider{}, false},
        {"local", "local", &LocalGitProvider{}, false},
        {"case insensitive upper", "GitHub", &GitHubProvider{}, false},
        {"case insensitive all caps", "GITHUB", &GitHubProvider{}, false},
        {"case insensitive local", "LOCAL", &LocalGitProvider{}, false},
        {"trimmed spaces", "  github  ", &GitHubProvider{}, false},
        {"unknown gitlab", "gitlab", nil, true},
        {"unknown bitbucket", "bitbucket", nil, true},
        {"empty string", "", nil, true},
        {"whitespace only", "   ", nil, true},
    }
    for _, tt := range tests {
        t.Run(tt.name, func(t *testing.T) {
            got, err := f.Create(tt.providerType, Credentials{Token: "test-token"})
            if tt.wantErr {
                assert.Error(t, err)
                assert.True(t, errors.Is(err, ErrUnknownProvider))
                assert.Nil(t, got)
                return
            }
            assert.NoError(t, err)
            assert.NotNil(t, got)
            assert.IsType(t, tt.wantConcreteType, got)
        })
    }
}

func TestNewFactory_ReturnsNonNil(t *testing.T) {
    f := NewFactory()
    assert.NotNil(t, f)
}
```
