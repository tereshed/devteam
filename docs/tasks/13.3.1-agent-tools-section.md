# 13.3.1 UI: секция «Инструменты» агента (tool bindings)

**Sprint:** 13 — Frontend: Настройки команды и проекта (Flutter)  
**Связано с:** **`docs/tasks/13.3-agent-edit-dialog.md`** (родительская задача; **tools** только там после выполнения API).

**Цель:** добавить в **`AgentEditDialog`** (или вложенный экран) выбор инструментов агента из данных бэкенда, **без** хардкода списка на фронте.

**Статус:** ⬜

---

## Канон (черновик до появления API)

| Тема | Решение |
|------|---------|
| Источник списка tools | Только **HTTP** с бэкенда; **запрещён** статический список в Dart. |
| Место в UI | Расширение **`agent_edit_dialog.dart`** или отдельный виджет в **`features/team/presentation/widgets/`**. |

**Критерии приёмки** и тесты дописать после фиксации контракта **`tool_definitions`** и ответа команды.
